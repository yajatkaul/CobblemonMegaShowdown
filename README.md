## **Features**

* Datapackable features
* Cosmetics support
* All form changes
* Riding supported
* All 48 Megas
* Accessories support
* Adds Primals
* Adds battle mechanics:

  * Tera gimmick
  * Mega Evolution
  * Z-Moves
  * Dynamax
  * Ultra Burst
* All fusion Pokémon
